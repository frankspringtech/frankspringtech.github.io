❶ Login to bitstamp on PC (www.bitstamp.net), click the account name, then click “View my account”

❷ Click “Security” - “API Access” in the left list

❸ Click “New API Key” to create a new API

❹ Select access permission, enter Two- factor Authentication code, then click “GENERATE KEY”

❺ Create successfully, fill in the API Keys and Secret Keys to CoinPlanet to track your portfolio
