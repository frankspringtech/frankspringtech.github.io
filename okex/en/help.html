❶ Login to OKEx on PC (www.okex.com), click your accout, then select “Settings”

               

❷ Select “My API” in the left list

               

❸ Click “Create new API”

❹ Select Trade for for reading portfolio, enter Display name and Google code, then “Confirm”

❺ Select “View” and enter verification code

❻ Create successfully, import API Key with QR code or manually type/paste in CoinPlanet to track your portfolio
